What are semantic elements?

Semantic elements provide information to web browsers and developers about the role and importance of different parts of the web page. Using semantic elements makes the code easier to read for browsers and humans. It also leads to more consistent code.

Consistent Code

Consistent code is important because there's many different ways to create certain elements. Sticking with one consistent way, such as using a header tag vs div class="header", makes it easier for everyone.
