HTML5 Semantic Elements

The purpose of this page is to understand and use HTML semantic elements properly.

What are semantic elements?

Semantic elements help provide additional information to web browsers and developers about the roles and importance of different parts of the web page. Using semantic elements makes the code easier to read, both to browsers and developers. It also leads to more consistent code.

Consistent Code

Consistent code is important because there's many different ways to create certain elements. Sticking with one consistent way, such as using a header tag vs div class="header", makes it easier for everyone.
